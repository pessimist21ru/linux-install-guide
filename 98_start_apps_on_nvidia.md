# Настройка запуска приложений на дискретной видеокарте NVIDIA

  После установки драйвера `nvidia-driver` на ноутбуках с гибридной графикой
(Intel + NVIDIA) возникает логичный вопрос: как запустить тяжёлое приложение
на дискретной видеокарте? В Debian нет утилиты `nvidia-prime` из Ubuntu,
но это не проблема — мы создадим свой простой и элегантный скрипт.

  Но для начала можно проверить работу установленного драйвера:

```bash
# Должны увидеть две карты
glxinfo | grep "OpenGL vendor"
# Если не установлен glxinfo-utils — поставь mesa-utils

# Протестируем запуск простого приложения на Nvidia
__NV_PRIME_RENDER_OFFLOAD=1 __GLX_VENDOR_LIBRARY_NAME=nvidia glxinfo | grep "OpenGL vendor"
# Должен выдать "NVIDIA Corporation"
```

  Создаём скрипт `/usr/local/bin/nvidia-run` и делаем его исполняемым.
Данный скрипт будет доступен для всех пользователей системы:

```bash
echo '#!/bin/bash
__NV_PRIME_RENDER_OFFLOAD=1 __GLX_VENDOR_LIBRARY_NAME=nvidia exec "$@"' | sudo tee /usr/local/bin/nvidia-run
sudo chmod +x /usr/local/bin/nvidia-run
```

  Что делает этот скрипт?
  Он просто подставляет перед запуском любой программы две специальные
переменные окружения:

    __NV_PRIME_RENDER_OFFLOAD=1 — включает механизм рендеринга на
    дискретной карте с выводом изображения через встроенную в процессор;

    __GLX_VENDOR_LIBRARY_NAME=nvidia — указывает системе использовать библиотеки
    OpenGL от NVIDIA.

  **Примечание по exec:** Команда `exec` заменяет процесс скрипта запускаемой
программой. Это экономит ресурсы (не остаётся висящего процесса bash).
Переменные окружения, заданные перед `exec`, передаются программе и всем её
дочерним процессам. Это особенно важно для приложений, которые запускают
свои «дочки» (например, 3D-viewer в KiCad).


## Как пользоваться

* Запуск из терминала:

  Вместо обычной команды просто добавьте перед ней nvidia-run:

```bash
# Обычный запуск
firefox-esr

# Запуск приложения на NVIDIA
nvidia-run firefox-esr

# Тест: сравните производительность шестерёнок
glxgears # Обычный запуск
nvidia-run glxgears # Запуск приложения на NVIDIA
```

* Запуск из меню:

В контекстном меню пункта меню приложений выбираем "изменить приложение"
и в поле "Программа" перед командой добавляем `nvidia-run`.

* При редактировании меню в строке `exec` также добавить `nvidia-run`
перед командой запуска. Например, копируем штатный ярлык приложения
в свою локальную директорию и редактируем его:

```bash
cp /usr/share/applications/firefox-esr.desktop \
~/.local/share/applications/firefox-esr.desktop
nvim ~/.local/share/applications/firefox-esr.desktop
```

  Находим строку начинающуюся с `Exec=` и приводим её к следующему виду:

```ini
Exec=nvidia-run /usr/lib/firefox-esr/firefox-esr %u
```

  После этого в меню приложений появится новый ярлык. Его можно переименовать,
например, в «Firefox (nv)», чтобы не путать с обычным.
