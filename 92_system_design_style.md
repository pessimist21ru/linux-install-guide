# Системное оформление: тема Gruvbox

  Создаём свое комфортное, не утомляющее глаза окружение, выдержанное в единой
палитре темы оформления, вдохновлённой

## Базовая информация

*   **Дистрибутив:** Debian 13 (Trixie)
*   **Окружение:** KDE Plasma 6.3.6 (KF 6.13.0, Qt 6.8.2, X11)
*   **Шрифты:** `Noto Sans` +1 ко всем размерам (это позволяет сохранить
        рабочее поле в CAD, но читать текст комфортно на разрешении 1920 x 1080
        с повышенным dpi, относительно полноразмерных мониторов, матрице
        ноутбука).
*   **Ночная цветовая схема:** Отключена (тёплые тона темы уже создают мягкий
        контраст).

## Пошаговая настройка (в порядке меню Параметров системы)

### 1. Оформление рабочей среды (Global Theme)
*   **Тема:** `Breeze, тёмный вариант`
*   **Действие:** Выбрать из стандартных, без установки глобальных тем. 
        Мы настраиваем всё вручную, чтобы не сломать систему скопом.

### 2. Цвета (Colors)
*   **Тема:** `Gruvbox plasma colors`
*   **Автор:** `adhe`
*   **Способ установки:** «Загрузить новые цвета...» → фильтр `gruvbox` →
        выбрать.
*   **Ссылка:** [KDE Store / adhe](https://store.kde.org/p/1327719)
        (основная страница автора)

### 3. Оформление приложений (Application Style)
*   **Движок:** `Fusion`
*   **Комментарий:** Обеспечивает аккуратные, скруглённые, чуть выпуклые
        элементы. В сочетании с палитрой Gruvbox даёт идеальный баланс.

### 4. Оформление рабочего стола (Plasma Style)
*   **Тема:** `GruvboxPlasma`
*   **Автор:** `adhe`
*   **Способ установки:** Ручной. Скачать архив с
        [KDE Store / p/1327719](https://store.kde.org/p/1327719)
        и распаковать в `$HOME/.local/share/plasma/desktoptheme/`.
*   **Действие:** Выбрать тему в списке.
*   **Доработка:** Шрифт виджета часов изменён на
        `Noto Sans ExtraCondensed Medium 10`.
*   **Сравнение с аналогами:** Тема от `3ximus`
        ([GitHub / 3ximus/gruvbox-plasma](https://github.com/3ximus/gruvbox-plasma))
        более контрастная и имеет лишние цветовые блоки. Версия `adhe`
        выбрана за гармоничность и лаконичность.

### 5. Оформление окон (Window Decorations / Aurorae)
*   **Тема:** `GruvboxAurorae`
*   **Автор:** `adhe`
*   **Способ установки:** Скачать архив
        [KDE Store / p/1327718](https://store.kde.org/p/1327718/)
        и распаковать в `$HOME/.local/share/aurorae/themes/`.
*   **Действие:** Выбрать тему в разделе «Оформление окон».

### 6. Значки (Icons)
*   **Тема:** `Gruvbox Plus Icon Theme Pack`
*   **Автор:** `eleuth`
*   **Способ установки:** Скачать архив с
        [KDE Store / p/1961046](https://store.kde.org/p/1961046)
        или найти через «Загрузить наборы значков...» и распаковать в `$HOME/.local/share/icons/`.
*   **Комментарий:** Превосходит тему от `adhe` по количеству
        отрисованных значков и общему качеству прорисовки.

### 7. Курсоры мыши (Cursors)
*   **Тема:** `Bibata-Modern-Amber`
*   **Автор:** `ful1e5`
*   **Способ установки:** Скачать с
        [GitHub / ful1e5/Bibata_Cursor](https://github.com/ful1e5/Bibata_Cursor)
        и распаковать в `$HOME/.local/.icons/`.
*   **Действие:** Выбрать тему в разделе «Курсоры».

### 8. Системные звуки (System Sounds)
*   **Тема:** `Океан` (стандартная).
*   **Рекомендация:** Оставить как есть, либо вручную заменить отдельные звуки
        на более спокойные из набора `freedesktop`.

### 9. Заставка (Splash Screen)
*   **Тема:** Одна из `QuarksSplash` (выбрана за минималистичный дизайн).
*   **Авторы и варианты:**
        `QuarksSplashLight` от `adhe`:
            [KDE Store / p/1304258](https://store.kde.org/p/1304258)
            — светлый вариант.
    *   `QuarksSplashDark` от `adhe`:
            [KDE Store / p/1304256](https://store.kde.org/p/1304256)
    *       — тёмный вариант.
    *   `QuarksSplashDarker` от `igflavius`:
            [KDE Store / p/1401423](https://store.kde.org/p/1401423)
            — ещё более тёмный вариант.
*   **Способ установки:** Распаковать архив в
        `$HOME/.local/share/plasma/look-and-feel/`, заменив файл
        `metadata.desktop` на `metadata.json` (необходимо для Plasma 6).
*   **Пример содержимого `metadata.json`:**
    ```json
    {
        "KPackageStructure": "Plasma/LookAndFeel",
        "KPlugin": {
            "Name": "QuarksSplashDark",
            "Description": "QuarksSplashDark",
            "Authors": [{"Name": "adhe", "Email": ""}],
            "License": "GPLv3",
            "Id": "QuarksSplashDark",
            "Category": ""
        },
        "X-Plasma-APIVersion": "2"
    }
    ```

### 10. Вход в систему (SDDM)

*   **Тема:** Debian Breeze (основа).

*   **Действие:** Фон изменён вручную. Необходимо создать или отредактировать
        файл /usr/share/sddm/themes/debian-breeze/theme.conf.user,
        добавив строки:
    ```ini

    [General]
    background=/путь/к/вашему/фону.png (только латиница в пути!)
    type=image
    ```

*  **Рекомендация:** Использовать тёмный, тёплый фон из нашей цветовой гаммы,
        чтобы не нарушать логику оформления.
