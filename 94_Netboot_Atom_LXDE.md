# Установка менеджера питания

  Для ноутбуков в графическом окружении отсутствует индикатор заряда
аккумулятора и подсистема управления питанием. В случае установки на
стационарный компьютер или при установки окружения кроме LXDE можно пропустить.
  При использовании других окружений, кроме LXDE, возможно уже присутствует в
системе и установку можно пропустить.

Устанавливаем вручную:

xfce4-power-manager          - менеджер питания
xfce4-power-manager-settings - графическая утилита для настройки менеджера
                                питания
xserver-xorg-input-synaptics - драйвер тачпада с возможностью настройки

//устанавливаем дополнительные службы, которых не хватает в LXDE
```bash
sudo apt-get install xfce4-power-manager* xserver-xorg-input-synaptics -y
```

//перенастраиваем графический сервер с новыми параметрами
```bash
sudo dpkg-reconfigure xserver-xorg
```

//настраиваем нажатие левой кнопки мыши по касанию на тачпаде
```bash
echo "@synclient TapButton1=1" | tee -a ~/.config/lxsession/LXDE/autostart
```

//перезагружаем систему, чтобы все службы нормально загрузились
```bash
sudo reboot
```


# Установка bluetooth

  При использовании других окружений, кроме LXDE, возможно уже присутствует в
системе и установку можно пропустить.

bluetooth - это метапакет, который содержит утилиты и службы для работы
            с bluetooth
bluez - стэк программ для работы с bluetooth:
    bluez-cups - драйверы для принтеров, подключаемых по bluetooth;
    bluez-meshd - демон (фоновый процесс) для bluetooth-сети;
    bluez-obexd - служба обмена данным;
    bluez-source - исходный код;
    bluez-test-scripts - тестовые скрипты.
blueman - графический менеджер bluetooth

Перед установкой нужно проверить работает ли служба bluetooth, так как 
графическое приложение для настройки отсутствует в LXDE. Возможно, что всё
работает и нужно установить только графическое приложение для настройки.

Для проверки нужно выполнить команду, которая запросит состояние службы. Если 
система ответит и покажет вывод с текущим состоянием, значит служба установлена
и работает.

```bash
sudo systemctl status bluetooth.service # проверка работы службы
sudo systemctl restart bluetooth.service # перезапуск службы
```

Если система выдаёт сообщение, что служба недоступна (неизвестная команда или
что-то подобное) необходимо установить bluetooth и bluez:

```bash
sudo apt install bluetooth bluez -y
```

После установки и запуска службы нужно установить графический менеджер blueman
для работы с устройствами и настройки не из консоли.
```bash
sudo apt install blueman -y
```


## Настройка джойстика


Для настройки джойстика в есть графический менеджер `jstest-gtk`.

```bash
sudo apt-get install jstest-gtk
```

Включаем джойстик, сопрягаем в blueman (для bluetooth джойстика), запускаем
`jstest-gtk` из "Меню -> Система" и проверяем работают ли кнопки.
