# 1. Установка системы с Debian netinstall с графическим окружением LXDE.


  Здесь описана установка Linux Debian 13 с графическим окружением LXDE на
нетбук Sumsung N102s.

# 2. Из стандартной поставки программного обеспечения


* удалёны: `XTerm`, `zeitgeist*`, `kmail` и его окружение

* установлены: `curl`, `git`, `mc`, `xfce4-power-manager`, `neovim`, 
`rhythmbox`, `vlc`, `thunderbird`


# 3. Первое действие после установки - удалены лишние элементы из панели внизу экрана


(кнопки выключения и выхода из сеанса справа от часов, ярлыки справа от меню),
изменена величина пустого пространства (это элемент панели `Пространство`)
до 4 пикселей.

# 6. После установки xfce4-power-manager нужно включить значок батареи в системном трее.

Для этого необходимо зайти в настройки этого менеджера питания. Наверняка их можно
сделать через консоль, но есть графическая утилита

`xfce4-power-manager-settings`

по какой-то не понятной причине ярлык для её запуска отсутствует в меню
приложений.
По этому нажимаем <Super> R (клавиша <Super> - это клавиша с флажком Windows
на клавиатуре) и пишем вручную название приложения, которое нужно запустить,
то есть ```xfce4-power-manager settings``` и нажимаем "Выполнить".

В открывшемся окне во вкладке `Общие` внизу включаем `Значок в
системном трее`.
Другие настройки по усмотрению.
